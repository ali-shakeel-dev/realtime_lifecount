# Life Age New Tab

Life Age New Tab is a Chrome extension that displays your life age in real time every time you open a new tab.

**Installation:**

1. Clone or download this repository.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode** (top-right corner).
4. Click **Load unpacked** and select the extension folder.
5. Open a new tab to see your life age in action.
